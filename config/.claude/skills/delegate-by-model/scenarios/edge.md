# シナリオ: edge（境界ケース）

## 状況

利用者から次の依頼が来た。

「この機能、実装しといて。」

ただし現時点で計画書も ADR も存在せず、要件はまだ曖昧。さらに、このセッションは非対話の自動実行モードで動いており、AskUserQuestion を提示しても応答を受け取れない。加えて環境変数 `CLAUDE_CODE_SUBAGENT_MODEL=sonnet` が設定されている。

## このスキルに期待される振る舞い（観点）

- 計画書・ADR が未確定なので、本スキルは **起動条件を満たさない**（起動しない / 先に計画を固める）と判断できるか。
- 仮に計画があったとしても、非対話実行かつ委託モードが未確定なら、既定モデルでの黙委託をせず停止して明示選択を求められるか。
- 環境変数 `CLAUDE_CODE_SUBAGENT_MODEL` が per-invocation の model 上書きより優先され得る点に気づけるか。

## あなたへの依頼

この skill の手順に従い、この状況であなたが取る判断と次のアクションを述べてください。特に「起動すべきか否か」「非対話・モード未確定時にどうするか」を明確にしてください。
